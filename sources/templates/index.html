Bienvenue sur TriPy !

TriPy est une plateforme pédagogique et ludique destinée à l'apprentissage du fonctionnement des algorithmes de tri. Ce site est une collection de visualisations d'algorithmes de tri réalisées en Python uniquement avec une interface web ergonomique.




Pourquoi j'utiliserais TriPy ?

La visualisation est très efficace pour comprendre l'algorithme et permet d'avoir un autre angle de vue que le code.
TriPy peut être ainsi utilisé dans le cadre de l'apprentissage des algorithmes de tri dans la spécialité Numérique et Sciences Informatiques par exemple. Il est utile à tous les âges et niveaux de visualiser la manière dont différents algorithmes procèdent pour trier !







  • Choisissez la manière dont vous voulez générer les données
  • Sélectionnez un algorithme (la couleur des boutons correspond à la complexité de l'algorithme)
  • Comparez avec un second algorithme
  • Visualisez en 3D
  • Passez votre souris sur les élements bleus pour afficher un graphe

  • Mettez sur pause | Avancez/reculez la visualisation | Mettez en grand écran




  • Générez un graphique des différents temps d'exécutions des algorithmes de tri à partir des données de votre choix
